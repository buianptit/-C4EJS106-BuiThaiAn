1. let vs var vs const :
1.1 Var và const trong javascript là gì ?
- Var để khai báo,khởi tạo 1 biến ( Thay đổi )
- Const để khởi tạo 1 hằng số ( Bất biến )
1.2 Sự khác biệt giữa let và var :
- Trong javascript,có 2 loại scope ( Phạm vi ):
- Nếu sử dụng var thì phạm vi của biến sẽ bị giới hạn trong function ( function - scope )
- Sử dụng let thì phạm vi của nó sẽ bị giới hạn bởi 1 block - scope ( cặp dấu {}) trong javascript.
1.3 Sự khác biệt giữa let và const :
- Phạm vi sử dụng của cả 2 đều là block - scope ( trong dấu {} ) tuy nhiên const không thể reasign ( Tái khai báo hay cập nhật lại giá trị mới để thay thế cho giá trị trước của biến )
1.4 Dùng trong trường hợp nào ?
- Let và var đều dùng để khởi tạo biến
- Const dùng để khai báo 1 hằng số
